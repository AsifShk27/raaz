# SAP-C02 Official Baseline (Verified 2026-02-10)

This reference is a baseline snapshot for coaching. Re-verify with official AWS pages when the user asks for latest updates.

## Exam Snapshot

- Exam: AWS Certified Solutions Architect - Professional (`SAP-C02`)
- Duration: 180 minutes
- Format: multiple choice and multiple response
- Question count interpretation:
  - Certification page: 75 questions total
  - Exam guide: 65 scored + 10 unscored questions
- Passing score: scaled 750 (Professional-level standard)
- Price: 300 USD

## Domain Weights (Current Guide)

- Domain 1: Design Solutions for Organizational Complexity (26%)
- Domain 2: Design for New Solutions (29%)
- Domain 3: Continuous Improvement for Existing Solutions (25%)
- Domain 4: Accelerate Workload Migration and Modernization (20%)

## Policy Facts

- If an attempt fails, retake is allowed after 14 calendar days.
- If an attempt passes, the same exam cannot be retaken for 2 years.
